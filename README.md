# TIC-80 Pro Manager

This is to install, upgrade, or uninstall the pro version of TIC-80 on Fedora.

Vibe coded with Gemini.

### Instructions
1. Download the latest release version
2. Open a terminal wherever you downloaded it (most likely your downloads folder)
3. Run "chmod +x tic-80-manager"
4. Run "./tic-80-manager"
